## üêõ Common Issues

**"File not found" error?**
- Check folder names match exactly
- Check file names match exactly
- Make sure files are in correct folders

**Compilation errors?**
- Check error message (usually tells you what's missing)
- Missing images can be commented out temporarily
- Check that main document is set correctly

**PDF too large?**
- This is normal with all code included
- PDF will be 50-100+ pages
- You can split into multiple files if needed
