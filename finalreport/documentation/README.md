# Bin Diesel Final Report Documentation

This folder contains all documentation, diagrams, and code for the final report.

## Structure

- `code/` - All source code with detailed comments
  - `main_control_system.py` - Main entry point (YELLOW)
  - `hardware_controllers/` - Motor, Servo, TOF sensor (RED)
  - `perception_modules/` - Wake word, YOLO, ArUco (YELLOW/ORANGE)
  - `computation_modules/` - State machine, optimizations (BLUE)
  - `configuration/` - Config file (GREEN)
- `diagrams/` - All system diagrams
- `documentation/` - Report sections
- `data/` - Generated test data and graphs

## Color Coding

- **YELLOW**: Main control system and ArUco detection
- **ORANGE**: YOLO pose tracking
- **RED**: Hardware controllers (Motor, Servo, TOF)
- **BLUE**: State machine and computation modules
- **GREEN**: Configuration
